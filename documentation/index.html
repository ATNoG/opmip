Introduction

The Open Proxy Mobile IPv6 (OPMIP) is an implementation of the Proxy Mobile IPv6 [1] (PMIPv6) standard.

The implementation of the PMIPv6 is provided by two applications:

  • opmip-mag - implements the Mobile Access Gateway (MAG)
  • opmip-lma - implements the Local Mobility Anchor (LMA)

[1] RFC 5213 - http://tools.ietf.org/html/rfc5213
